# Project Iris

Curriculum Auto-Alignment & Assessment Ingestion Tool for BC, AB, and ON.

## Features

- **Curriculum Standards Management**: Browse and search curriculum standards by province
- **Assessment Ingestion**: Upload district assessments for alignment analysis
- **AI-Powered Alignment**: Automatic curriculum standard mapping using Claude API
- **Validation & Audit**: User validation with full audit trails
- **Multi-Province Support**: British Columbia, Alberta, Ontario

## Tech Stack

- **Frontend**: Next.js 14 (React)
- **Backend**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **AI**: Anthropic Claude API
- **Deployment**: Vercel

## Setup

### Prerequisites

- Node.js 18+
- Supabase account
- Anthropic API key
